# 🎂 Happy Birthday Project

Proyek spesial untuk merayakan ulang tahun dengan perpaduan desain retro **Gameboy**, efek **3D**, dan **Kembang Api** yang interaktif.

## ✨ Fitur Utama
- 🎮 **Tema Gameboy**: Navigasi unik menggunakan antarmuka konsol genggam klasik.
- 🎆 **Efek Kembang Api**: Animasi kembang api yang memukau untuk memeriahkan suasana.
- 🎵 **Latar Musik**: Transisi musik yang mulus untuk pengalaman yang lebih emosional.
- 🌎 **Visualisasi 3D**: Elemen visual interaktif berbasis Three.js.
- 📱 **Responsif**: Dapat diakses dengan baik melalui perangkat mobile maupun desktop.

## 🚀 Cara Menjalankan
1. Pastikan Anda memiliki browser modern terinstall.
2. Clone repositori ini:
   ```bash
   git clone https://github.com/naansa-naufalsaputra/Happy-Birthday.git
   ```
3. Buka file `index.html` langsung di browser atau gunakan live server (Node.js `serve` atau ekstensi VS Code).
   ```bash
   npx serve .
   ```

## 🛠️ Teknologi yang Digunakan
- **HTML5 & CSS3**: Struktur dan styling (Vanilla CSS).
- **JavaScript (Vanilla)**: Logika aplikasi dan interaktivitas.
- **Three.js**: Untuk render elemen 3D.
- **Gameboy UI Design**: Elemen visual retro.

## 📂 Struktur Folder
- `index.html`: Halaman utama aplikasi.
- `styles.css` & `gameboy-styles.css`: File styling.
- `gameboy-script.js`: Logika khusus untuk interaksi tema Gameboy.
- `js/`: Folder berisi library atau script pendukung (seperti fireworks).
- `assets/`: Media pendukung (gambar, musik, dll).
